# 💻 Dev Portfolio

A multi-page personal portfolio website with a terminal-inspired dark theme.

## Pages
- **Home** — Hero, stats, featured projects
- **About** — Bio, experience timeline
- **Projects** — Filterable project grid + live GitHub repos via API
- **Skills** — Animated progress bars, tech stack
- **Blog** — Post listings
- **Contact** — Form + social links

## Tech
- Vanilla HTML, CSS, JavaScript (no frameworks)
- Google Fonts — JetBrains Mono + Syne
- GitHub REST API for live repo fetching

## Usage
1. Clone the repo
2. Replace `devuser` with your GitHub username in `index.html` and `projects.html`
3. Update name, bio, and links across pages
4. Open `index.html` in a browser — no build step needed

## Structure
├── index.html
├── css/          # Per-page stylesheets + global style.css
├── js/           # main.js + projects.js
└── pages/        # about, projects, skills, blog, contact
